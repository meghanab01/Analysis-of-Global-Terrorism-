# Analysis-of-Global-Terrorism-
Analysis of Global Terrorism Database (GTD), exploring temporal trends, regional hotspots, attack types, and human impact across 50 years

---

### Dataset

> Due to data usage restrictions, the dataset is **not included** in this repository.
> You can request access and download the dataset from the official [Global Terrorism Database](https://www.start.umd.edu/gtd/) website.

---

### Tools & Libraries Used

* **Python** (pandas, numpy, matplotlib, seaborn)
* **Jupyter Notebook**
* Data Cleaning & Feature Analysis
* Exploratory Data Analysis (EDA) & Visualization

---

### Key Insights

* **Peak Terrorism Years:** 2014–2016 saw the highest number of incidents globally.
* **Regional Hotspots:** Middle East & Sub-Saharan Africa dominate attack frequencies.
* **Attack Patterns:** Bombings and armed assaults remain the most prevalent methods.
* **Human Impact:** While attacks peaked in the 2010s, lethality was highest in the 1980s.
* **Temporal Trends:** Attack rates declined near 2020s, possibly due to counter-terrorism tech.

### Conclusion

Terrorism has evolved from localized insurgencies to globally connected networks.
While modern decades see more attacks, enhanced surveillance and counter-terrorism efforts have gradually reduced success and lethality rates — marking a shift in both **tactics and effectiveness**.
